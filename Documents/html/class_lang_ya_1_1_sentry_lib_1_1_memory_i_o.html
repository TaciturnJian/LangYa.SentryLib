代表一个基本的读写类，在读和写的时候使用

LangYa::SentryLib::MemoryView
LangYa::SentryLib::MemoryView
代表一个内存的视图，保存内存的头指针和长度，供其他读写。 不能保证作为传入的参数时得到的内存视图是有效的，因此作为函数参数时，这个类的基础是对调用者的信任。
Definition MemoryView.hpp:14

. 一般来说，称读取操作为 Read/Receive ，写入操作为 Write/Send 。

virtual MemoryView::SizeType LangYa::SentryLib::MemoryIO::Read ( const MemoryView &  view)
pure virtual

读取字节（最多读取 view 的所有字节）。

参数
view缓冲区的内存视图，读取到的字节将存放在这个视图对应的内存中。
警告
此函数的实现一般不会检查视图是否有效，所以调用者必须自己保证视图有效。
返回
读取并存放了多少个字节。
