# SentryLib 代码文档

SentryLib 是一个机器人工具库，简单来说，包含了如下功能或接口：

- 内存管理（缓冲区，字节保护）
- 序列化与反序列化接口
- 连接接口及串口连接、TCP套接字连接实现
- 机器人控制器和传感器接口
