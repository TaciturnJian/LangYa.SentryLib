template<typename TClient>
class LangYa::Network::IClientAcceptor< TClient >

代表可以接收客户端的类。

该类的实现类可以接收客户端的连接请求，并且在同一时间最多返回一个有效客户端。 通过反复调用 AcceptClient 即可实现接收多个客户端连接。
