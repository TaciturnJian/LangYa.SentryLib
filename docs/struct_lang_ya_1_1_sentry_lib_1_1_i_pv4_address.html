详细描述

代表 IPv4 地址。

本质上是一个 unsigned char[4] ，但是为了方便使用，所以封装了一下。 顺序是 A.B.C.D，不包含端口，如果需要端口请使用 endpoint 。

◆ Parse() [1/2]

bool LangYa::SentryLib::IPv4Address::Parse ( std::istream &  stream)

从输入流中读取连续四个 unsigned char 作为 IPv4 地址。

找到四个无符号字符整数返回 true ，否则，将会返回 false 。 例如 "192l168?137:1" 和 "192.168.137.1" 都会被解析成 unsigned char[4]{192, 168, 137, 1} 。
