virtual bool LangYa::SentryLib::IOConnection::Open ( )
pure virtual

打开此连接，成功打开后便可以开始读写。

在未开启连接之前，调用 Read 和 Write 方法是未定义行为。 调用此函数往往会阻塞当前线程，直到连接建立成功返回 true 或判断无法建立连接后返回 false 。
