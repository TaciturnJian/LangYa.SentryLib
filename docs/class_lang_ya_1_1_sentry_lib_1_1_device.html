详细描述

代表一种虚拟的设备，它需要有一个外部循环去调用。 大多数使用情况都是在一个线程中调用 Tick() 方法。

◆ Tick() [2/2]

virtual void LangYa::SentryLib::Device::Tick ( )
pure virtual

激活设备使其运作一次 如果设备未开启，则此函数应当使设备开启，如果成功则开始运作。 如果运作发生严重错误则抛出异常。 如果设备运作一次的时间较长，则此函数会阻塞当前线程。 如果你希望此设备保持运行，那么使用一个循环是最佳选择，简单的代码如下：

//#include <thread> //引用此头文件，让当前线程可以休眠
while (true) {
using namespace std::chrono_literals;
std::this_thread::sleep_for(1ms); //设置最大频率为 1000Hz
device.Tick();
}

或者你可以自己实现一个稳频器。

在 LangYa::SentryLib::DeserializableContentSensor< TSensorData > , 以及 LangYa::SentryLib::SerializableContentController< TControllerData > 内被实现.
