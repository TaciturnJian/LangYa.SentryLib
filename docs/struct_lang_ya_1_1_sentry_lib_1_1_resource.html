详细描述

template<typename TContent>
struct LangYa::SentryLib::Resource< TContent >

Represent a resource with flag (usage indicators), usually this resource is used in multi-threads. If you want to get the content, use '=' operator to copy the memory. The memory may be reading/writing in another thread!

模板参数
TContentThe type of content, must be trivially copyable.
