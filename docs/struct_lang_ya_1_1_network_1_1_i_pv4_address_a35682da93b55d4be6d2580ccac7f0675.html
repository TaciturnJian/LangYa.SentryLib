从输入流中读取连续四个 std::uint8_t 作为 IPv4 地址。

找到四个无符号字符整数返回 true ，否则，将会返回 false 。 例如 "192l168?137:1" 和 "192.168.137.1" 都会被解析成 std::uint8_t[4]{192, 168, 137, 1} 。
