DynamicStack 动态堆栈

动态堆栈代表一个内存空间，帮助大量使用共享指针的类管理内存。 交给动态堆栈的指针为共享指针，会在所有此共享指针被销毁时释放指向的内存。

动态堆栈的使用方法

++
DynamicStack<T> stack{};
auto shared_element = std::make_shared<T>();
auto& space = stack.RestoreSpace();
space = shared_element;
