VisonSensor 视觉传感器，提供机器人视觉相关信息。

VisonSensor 为机器人提供了额外的信息源，目前它能获取如下信息：

  • 提供视觉传感器上捕获到的装甲板的原始信息
  • 因为工程需要，添加了额外的一个接口，用于获取兑换框信息。
