Navigator 导航模块

Navigator 是自动机器人中的感知器和决策器的重要部分，它提供了如下功能

  • 感知自身位置，给出一个三维 float 坐标表示当前位置。
  • 感知周围障碍物，给出一系列瞬时二维 float 坐标表示障碍物的位置。
  • 给定目标点，给出从当前状态来看的最佳瞬时速度
警告
Navigator 不提供任何运动控制功能，它只是一个决策器，它的输出需要由运动控制器来执行。

Vector2F &  velocity 
)
virtual

给定目标地点，获取到达目标地点的速度。

使用此函数将设置导航器的目的地为 destination 。 之后，导航器会做一系列计算，并返回到达目的地的速度。

警告
返回 false 并不能完全代表导航器解算速度失败，可能是解算太慢等原因。 如果需要判断此函数当前不可用，请在一段时间内连续调用此函数。 连续返回 false 才说明导航器真正寄了。
参数
destination传入目的地的 ID 。
velocity用于接收速度的变量。
返回
如果解算出来了速度，则返回 true ，否则返回 false。
