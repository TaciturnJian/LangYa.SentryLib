Navigator 导航模块

Navigator 是自动机器人中的感知器和决策器的重要部分，它提供了如下功能

  • 感知自身位置，给出一个三维 float 坐标表示当前位置。
  • 感知周围障碍物，给出一系列瞬时二维 float 坐标表示障碍物的位置。
  • 给定目标点，给出从当前状态来看的最佳瞬时速度
警告
Navigator 不提供任何运动控制功能，它只是一个决策器，它的输出需要由运动控制器来执行。
