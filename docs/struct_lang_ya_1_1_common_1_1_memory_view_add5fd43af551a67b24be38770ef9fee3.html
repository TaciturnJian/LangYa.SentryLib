LangYa::Common::MemoryView::MemoryView ( auto *const  head,
const SizeType &  count = 1 
)
inlineexplicit

使用给定的内存头和内存长度构建一个内存视图。 如果提供的两个参数是非法的，那么此内存视图也是非法的，而且此类不会在构建时抛出异常。小心在之后的使用中出现段错误。

参数
head数组的头，它的元素类型将会自动推断，注意内存长度是 sizeof(*head) * count。
count数组的元素数量，注意内存长度是 sizeof(*head) * count。
警告
在使用时，经常会想当然认为第二个参数是内存的长度，这是错误的，它应该是数组的元素数量。

引用了 MemoryView().

被这些函数引用 LangYa::Common::PingPongBuffer::Exchange(), LangYa::Common::PingPongBuffer::ExchangeWithExchangeBuffer(), MemoryView(), LangYa::Common::UniqueBuffer::operator=() , 以及 LangYa::Common::UniqueBuffer::UniqueBuffer().
